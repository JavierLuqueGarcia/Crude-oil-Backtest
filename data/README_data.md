# data/ — Raw and Processed Data

## Files

### Raw Data (from authoritative sources)

- **prices_full.csv**
  - Source: Yahoo Finance (tickers CL=F for WTI, BZ=F for Brent)
  - Period: 2015-01-02 to 2026-07-10
  - Columns: Date, WTI, Brent
  - Rows: 2,896 (one per trading day)
  - Note: Uses front-month futures closing prices

- **eia_inventory.csv**
  - Source: U.S. Energy Information Administration, Weekly Petroleum Status Report
  - Series: WCESTUS1 (US Commercial Crude Oil Ending Stocks, weekly)
  - Columns: Date, Inventory_kbbl (thousands of barrels)
  - Notes: Excludes Strategic Petroleum Reserve; released Wednesdays for the prior week's Friday end
  - Updated weekly (last week: ~411M barrels as of Jul 2026)

- **eia_refutil.csv**
  - Source: EIA, Weekly Petroleum Status Report
  - Series: WPULEUS3 (US Refinery Utilisation Rate, weekly)
  - Columns: Date, RefUtil_pct (percentage of capacity)
  - Normal range: 80–95%
  - Updated weekly

### Derived Data

- **trades.csv**
  - Generated by running the backtest
  - Columns: entry_date, exit_date, entry (price), exit (price), pnl (%)
  - Rows: 59 (all completed trades from 2015–2026)
  - Use this to audit every individual trade

## How to Update

### Fresh Price Data

1. Go to https://finance.yahoo.com
2. Search CL=F (WTI)
3. Click "Historical Data"
4. Download as CSV for the full period you want
5. Paste into backtest/Crude_Oil_Backtest.xlsx, Data sheet, column B (WTI_Close)
6. Repeat for BZ=F (Brent) → column C

### Fresh Inventory & Refinery Data

1. Go to https://www.eia.gov/petroleum
2. Find "Weekly Petroleum Status Report"
3. Crude oil ending stocks (WCESTUS1): download Excel or use the API
4. Refinery utilisation (WPULEUS3): same location
5. Paste into data/eia_inventory.csv and data/eia_refutil.csv
6. Re-run the backtest (formulas in backtest/ will recalculate)

## Data Quality Notes

- **Prices:** Checked against official NYMEX closing data. No gaps (markets closed only on weekends/holidays).
- **Inventory:** EIA publishes every Wednesday afternoon EST (~4:15 PM) for the prior Friday. Values can be revised slightly the following week (preliminary → revised → final).
- **Refinery:** Calculated as a moving average by EIA; can fluctuate with maintenance schedules, hence seasonal patterns (summer = high, winter = lower).
- **Extreme values:** 
  - WTI fell to −$37.63 on April 20, 2020 (storage full, COVID demand collapse). This is real data, not an error.
  - Inventory peaked at 465.7M barrels (early May 2026) during the spring storage-building season.

## For Reproducibility

All three CSV files are sufficient to:
1. Regenerate the Excel backtest (paste into Data sheet)
2. Recompute all figures (Python scripts in src/)
3. Audit every trade (trades.csv lists all 59)

No proprietary data or feeds required. Everything is free and public.
